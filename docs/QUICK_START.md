### Partner Payouts
Each partner card shows:
- Partner name
- Hours worked
- Calculation breakdown
- Final payout amount
- Bill breakdown (how many $20s, $10s, $5s, $1s)

### Bills Needed
Total count of each bill denomination needed for all payouts.

**OCR not detecting partners?**
- Check image quality
- Ensure names and hours are on same line
- Try a different image format
- Make sure text is clear and readable

**Wrong calculations?**
- Verify the tip amount is correct
- Check that all partners were detected
- Review the hourly rate calculation

**Page not loading?**
- Clear browser cache
- Try a different browser
- Check internet connection (for first load)
